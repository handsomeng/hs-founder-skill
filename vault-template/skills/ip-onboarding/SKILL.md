# 采访 AI：建立人设档案

## 目标

通过对话采访，产出三份档案存进 `00-人设中枢/`：

1. `人设一页纸.md`：他是谁、做给谁看、要达到什么效果
2. `语言指纹.md`：他怎么说话，用于让 AI 产出像他本人
3. `观点库.md`：他反复强调的核心观点，持续追加

## 采访规则（重要）

- **一次聊一整个板块，不逐题问**。每个板块开始时，用一段话告诉他现在聊什么、为什么聊，然后把这个板块想了解的东西揉成 3 到 5 个提示点一次性说给他，请他想到哪说到哪，顺序随意，说漏了也没关系
- 他说完之后，你**当场总结**：「我听到的是……」，两三句话复述提取到的要点，让他确认或纠正
- 总结完，只对**致命缺口**补问，每个板块最多补 1 到 2 个问题。致命缺口的标准：缺了这条，下游选题和脚本就没法干活（比如受众完全没提、商业目的没说）。锦上添花的细节不追，留给复访
- 他说到具体故事、金句、独特经历时，原话记下来，这是最值钱的素材
- 全程口语化，像一个懂行的朋友在聊天，别像填表
- 整场 30 分钟左右的量。他累了随时可以说「先到这」，你保存已有内容，下次从断点继续

## 采访提纲（五个板块）

下面每个板块列的问题是**给你的提取清单**，用来对照他的自由发挥里哪些有了答案、哪些是致命缺口。别照着念，开场时把它们揉成那 3 到 5 个提示点。

### 板块一：基本盘

1. 用三句话介绍一下你自己和你的公司，就当我完全不认识你
2. 你在这个行业干了多少年？入行的契机是什么？
3. 你的公司现在主要靠什么赚钱？客户是谁？
4. 做这个 IP，你最直接的商业目的是什么？（获客 / 招人 / 融资 / 行业话语权 / 卖货）
5. 你打算主攻哪个平台？每周能拿出多少时间拍或写？

### 板块二：受众

6. 你最想让谁刷到你的内容？描述一个具体的人（职业、年龄、他正在为什么发愁）
7. 这个人现在关注谁？为什么那些人满足不了他？
8. 他看完你的内容，你希望他做什么动作？（关注 / 私信 / 转发给老板 / 下单）
9. 有没有你明确不想吸引的人群？

### 板块三：差异化弹药（最重要的板块，多挖）

10. 你经历过什么别人没经历过的事？（踩过的大坑、赚到的第一桶金、行业内幕）
11. 你有什么和行业主流唱反调的观点？越具体越好
12. 客户或朋友最常找你问什么问题？说明市场认你什么
13. 讲一个你最得意的案例，从头到尾讲一遍
14. 讲一个你最惨痛的失败，当时怎么想的，后来怎么翻的
15. 如果只能给同行一句忠告，你会说什么？

### 板块四：语言指纹

16. 你平时跟员工开会，说话是什么风格？（直接骂 / 讲故事 / 摆数据 / 开玩笑）
17. 你有什么口头禅或者常用的比喻？
18. 发一段你平时的微信语音转文字，或者随便对我说两分钟你对行业的看法（原样记录进语言指纹）
19. 你看到哪种自媒体腔调会觉得恶心、绝对不想变成那样？
20. 有什么词或说法是你绝对不会用的？

### 板块五：边界与目标

21. 有什么话题绝对不能碰？（竞对、前合伙人、政策敏感、家庭）
22. 公司和客户的信息，什么能说什么不能说？
23. 三个月后，这个 IP 做到什么程度你觉得「值了」？给个可衡量的标准
24. 你最担心做 IP 会带来什么麻烦？

## 产出格式

采访结束后（或他喊停时），生成三份文件：

**`00-人设中枢/人设一页纸.md`**：

```
# 人设一页纸
更新日期：YYYY-MM-DD

## 一句话定位
（他是谁 + 为谁 + 提供什么独特价值，一句话）

## 商业目的
## 目标受众画像
## 受众看完要做的动作
## 差异化资产（经历、案例、反共识观点，列清单）
## 三个月目标
## 红线清单（绝对不碰的话题和说法）
```

**`00-人设中枢/语言指纹.md`**：

```
# 语言指纹
更新日期：YYYY-MM-DD

## 整体风格（一段描述）
## 口头禅与常用比喻（原话列表）
## 原话样本（板块四「说两分钟看法」的原样记录）
## 禁用词与禁用腔调
```

**`00-人设中枢/观点库.md`**：

```
# 观点库
每条格式：## 观点原话 + 背后的故事/论据 + 记录日期
（采访中挖到的每个观点单独一条，之后持续追加）
```

生成后，把三份文件的核心内容念给他确认，哪里不对当场改。最后告诉他：「档案建好了，以后你随时说『提取选题』或『出框架』，我都会先看这份档案再干活。」

## 复访模式（他说「更新人设」时）

不重新采访，只问三个问题：

1. 最近有什么新的观点或者被验证/推翻的旧观点？
2. 最近有什么新的案例或故事？
3. 定位、受众、目标有变化吗？

把新内容追加进对应文件（观点库直接加条目，另两份更新对应段落并改更新日期），不动旧内容。
