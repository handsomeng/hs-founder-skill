# hs-founder-skill：创始人 IP 内容系统

## 简介

很多创始人想做 IP，卡在同一个地方：不缺经历和观点，缺一套能持续产出的系统。请代运营，写出来的东西不像自己；自己写，坚持不了两周。

这套 skill 把内容生产拆成五个环节，AI 负责结构、提炼、把关，你只做机器替代不了的事：说出你的真实经历，填上你的真实案例。

- **零技术门槛**：装好后只需要记五句话，全程和 AI 说话完成
- **像你，不像 AI**：先花 30 分钟采访你，生成人设档案（定位、语言指纹、观点库），之后所有产出都对着档案校准
- **刻意留白**：脚本只出骨架，核心案例强制留给你自己填，防止内容变成千篇一律的 AI 腔
- **越用越准**：素材、选题、成稿、数据全部沉淀在本地 Obsidian 知识库里，git 自动保底

## 新手指南

三步跑起来：

1. **装**：按下面的安装说明，一句话让 AI 装好
2. **聊**：说「开始采访」，和 AI 开放式聊 30 分钟，生成你的人设档案
3. **跑**：说「提取选题」让 AI 从你的日常素材里挖题，挑一个说「出框架」，填完留白说「审稿」

就这么简单。

## 安装

前提：电脑装好 git 和 Codex CLI（或 Claude Code）。启动 AI，说一句：

> 安装 https://github.com/handsomeng/hs-founder-skill 并配置我的 Obsidian 知识库

AI 会读到下方指令自动完成，最后引导你打开 Obsidian 并重启会话。

<details>
<summary>给 AI 的安装指令（用户说出上面那句话时照此执行）</summary>

自动完成以下全部步骤，过程中不问技术问题：

1. 问用户一个问题：「知识库文件夹想叫什么名字？（不说就默认叫 我的IP）」
2. `git clone https://github.com/handsomeng/hs-founder-skill.git /tmp/hs-founder-skill`
3. 把 `/tmp/hs-founder-skill/vault-template` 复制为 `~/<用户起的名字>`（家目录直接子目录；macOS 上禁止放 Downloads / Documents / Desktop，系统会拦 AI 读写）
4. 在新文件夹里 `git init && git add -A && git commit -m "init"`
5. 删除 `/tmp/hs-founder-skill`
6. 用大白话告诉用户剩下两个必须人工做的动作：
   - 打开 Obsidian，「打开文件夹作为仓库」，选刚建的文件夹
   - 退出当前 AI 会话，进入该文件夹重新启动（`cd ~/文件夹名` 后运行 codex 或 claude），然后说「开始采访」

</details>

## 工具清单

| 模块 | 口令 | 干什么 |
| --- | --- | --- |
| 采访建档 | 「开始采访」 | 五个板块开放式聊 30 分钟，生成人设一页纸、语言指纹、观点库。每月说「更新人设」增量补充 |
| 选题提取 | 「提取选题」 | 扫描收件箱素材，对照人设打分（120 分制：人设匹配 40 + 爆款潜力 60 + 时效 ±20），高分入选题库 |
| 脚本框架 | 「出框架」 | 按七步说服链出骨架：钩子给 3 个备选，核心案例留白等你填，全框架不超 500 字 |
| 审稿 | 「审稿」 | 三道检查：结构完整性、像不像你本人、去 AI 味。红黄绿分级出清单，只挑刺不代写 |
| 知识库管家 | 「收纳」「帮我找」「整理知识库」 | 任何资料自动归档建链，全库问答带出处，每周体检（收件箱清零、观点冲突、选题保鲜） |

## 使用主线

```
开始采访 ──→ 人设档案（地基，一次建好，每月更新）
                │
日常素材扔收件箱 ──→ 提取选题 ──→ 你拍板 ──→ 出框架 ──→ 你填留白 ──→ 审稿 ──→ 发布
                                                                        │
                                                          数据回填复盘，反哺下轮选题打分
```

## 使用场景

- **创始人本人**：每天语音随想扔收件箱，每周挑题出片，IP 内容不断更
- **给老板做 IP 的操盘手**：装机 + 陪跑第一次采访，之后老板自己就能用
- **咨询 / 代运营团队**：当作交付物装给客户，方法论固化在 skill 里，服务可复制
- **想学方法论的同行**：五个 SKILL.md 就是完整的创始人 IP 内容 SOP，直接读
